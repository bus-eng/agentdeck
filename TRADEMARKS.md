# AgentDeck Trademark Policy

## Ownership

The name "AgentDeck", the AgentDeck logo, icons, visual identity,
and all related branding elements are trademarks of
**Roberto Bustamante (virela-dev)**.

## What You May Do

- Refer to AgentDeck truthfully in your own projects (e.g., "compatible
  with AgentDeck", "built for AgentDeck").
- Use the name AgentDeck to describe the unmodified official software.
- Fork the code under AGPL-3.0-or-later terms **provided you remove or
  replace all AgentDeck branding**.

## What You May Not Do

- Distribute a modified version under the name "AgentDeck" in a way
  that suggests it is the official version.
- Use the AgentDeck name or logo to promote a hosted service without
  explicit written permission.
- Register domain names, package names, social media handles, or any
  identifier that is confusingly similar to "AgentDeck".
- Use the AgentDeck visual identity (logo, icons, color scheme) in a
  derivative project without removing all identifiable AgentDeck branding.

## Reporting Concerns

If you see someone using the AgentDeck name or branding in a way that
violates this policy, please open a GitHub issue or contact the
maintainer at the project's repository.

---

This policy is based on the model trademark guidelines used by many
open source projects. It is intended to protect the project's identity
without restricting legitimate use of the software.
